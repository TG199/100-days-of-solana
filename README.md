# 100 Days of Solana Challenge 🚀

Welcome to my **100 Days of Solana** challenge repository!  

This challenge is my commitment to consistently learning, building, and sharing my journey in the Solana ecosystem over the next 100 days as part of the **Major League Hacking (MLH)** challenge.

## About the Challenge

The goal of this challenge is to:
- Learn Solana blockchain development from the ground up
- Build projects and experiments using Solana tools and technologies
- Explore smart contracts/programs, wallets, dApps, and the Solana ecosystem
- Improve my Rust and Web3 development skills
- Share progress publicly and stay accountable

## Technologies & Tools

Throughout this challenge, I’ll be working with technologies such as:

- Rust
- Solana CLI
- Anchor Framework
- TypeScript / JavaScript
- Web3.js
- Phantom Wallet
- React / Next.js
- Devnet & Local Validator
